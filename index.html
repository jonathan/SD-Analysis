Sd-analysis

Sorts through experimental data and populates a database with data of interest

View project onGitHub

This is a ruby project to parse and clean up data from a laser analysis of SerDes chips. The cleaned up data is then inserted into a PostgreSQL DB and various SQL queries are run to analyze the data.
