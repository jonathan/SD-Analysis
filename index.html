Sd-analysis

Sorts through experimental data and populates a database with data of interest

View project onGitHub

This is a Ruby project to parse very large sets of raw data collected during single-event effect laser analysis of a serializer/deserializer (SerDes) integrated circuit. The script inserts the analyzed data into a PostgreSQL DB. SQL queries are utilized for final analysis of the data. This effort was necessary to determine the timing of the laser strike within a SerDes clock cycle. This is the first use of Ruby/SQL to analyze a data set of this magnitude for this application.

The linked paper was published in the IEEE Transactions on Nuclear Science. Section III.B. describes the analysis in detail.
